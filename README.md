# Dota 2 Draft Assistant 🎮

Автономный драфт-ассистент для Dota 2 с рекомендациями на основе контрпиков, синергии и меты.

## ⚡ Быстрый старт

```bash
# 1. Установи зависимости
pip install -r requirements.txt

# 2. Обнови данные (один раз, потом по необходимости)
python main.py --update

# 3. Запусти приложение
python main.py
```

## 🛠️ Установка

### Windows

1. **Установи Python 3.8+** с [python.org](https://python.org)

2. **Установи Tesseract OCR** (для будущего OCR):
   - Скачай с [GitHub](https://github.com/UB-Mannheim/tesseract/wiki)
   - Установи и добавь в PATH

3. **Клонируй/скачай проект**:
```bash
cd draft_assistant
```

4. **Установи зависимости**:
```bash
pip install -r requirements.txt
```

5. **Обнови данные** (первый запуск):
```bash

```

## 📊 Скоринг-модель

Формула расчёта:
```
total_score = base_winrate * w_base 
            + counter_bonus * w_counter 
            + synergy_bonus * w_synergy 
            + role_penalty * w_role 
            + meta_score * w_meta
```

Веса настраиваются в `config/settings.py`:
- `base_winrate` — базовый винрейт (вес 1.0)
- `counter_bonus` — бонус контрпиков (вес 0.8)
- `synergy_bonus` — синергия команды (вес 0.6)
- `role_penalty` — пенальти за дублирование ролей (вес 0.5)
- `meta_multiplier` — множитель меты (вес 1.0)

## 🔧 Настройка под себя

### Добавь свои синергии

В `scoring/recommender.py` метод `_get_synergy_pairs()`:

```python
# (hero_id_a, hero_id_b): bonus_score
(97, 18): 15,  # Magnus + Sven = +15 к скору
```

### Измени веса

В `config/settings.py`:

```python
SCORING_WEIGHTS = {
    "base_winrate": 1.0,
    "counter_bonus": 1.2,  # Увеличь важность контрпиков
    "synergy_bonus": 0.8,
    "role_penalty": 0.5,
    "meta_multiplier": 0.7,
}
```

## 🐛 Известные проблемы

1. **OCR требует настройки ROI** — каждое разрешение экрана требует своих координат
2. **OpenDota rate limits** — ~1 запрос/сек, используй кэш
3. **Данные могут устаревать** — обновляй `--update` после патчей
